# MoO Design System

Member Experience Design System for Mutual of Omaha.

## Design System Registry

**[`ds-registry.md`](ds-registry.md)** is the single source of truth for all design tokens, component keys, and icon keys. Reference it before creating or updating any wireframes or prototypes.
