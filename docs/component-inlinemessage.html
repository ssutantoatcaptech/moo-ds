Props

PropTypeDefaultDescription
property1 "Success" | "Error" | "Warning" | "Information" "Success" Controls the variant: color, icon, and tint.
title string — Bold heading text displayed in the message.
children ReactNode — Body content including text and optional links.
onDismiss () => void — Callback when the close button is clicked. If omitted, close button is hidden.
className string — Additional CSS class for the wrapper.

Anatomy

1. Status Stripe — 40px wide colored sidebar with centered icon (white). Rounded on left side (12px).
2. Content Area — Tinted background with matching border. Rounded on right side (12px). Contains title, body text, and optional links.
3. Title — Headline/H6 style (Source Sans Pro SemiBold, 14px, #444).
4. Body Text — Body/Smaller Text (Source Sans Pro Regular, 14px, #444) with optional inline links (#105fa8, underlined).
5. Close Button — Dismiss icon (top-right), optional via onDismiss prop.
